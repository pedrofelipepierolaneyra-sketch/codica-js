npm init -y

Este comando crea automáticamente un archivo package.json con información básica sobre tu proyecto.

2️⃣ Abre el archivo package.json en tu editor de código (por ejemplo, VS Code) y reemplaza su contenido por este:

{
        "name": "codica-js", "version": "1.0.0", "description": "Mi primer proyecto en Códica", "main": "index.js", "type": "module", "scripts": { "start": "node index.js" }, "author": "Tu Nombre", "license": "MIT" }
        
        
3️⃣ Asegúrate de tener un archivo index.js con el siguiente contenido:
 
console.log("¡Hola, Códica!");
 
4️⃣ Ejecuta tu archivo con el comando:
 
npm start
 
Deberías ver este mensaje en la terminal:
 
¡Hola, Códica!
 
5️⃣ Sube los cambios a GitHub:
 
🔸 Agrega todos los archivos al control de versiones:
 
git add .
 
🔸 Haz un commit con un mensaje descriptivo:
 
git commit -m "Inicialización del proyecto con package.json y ejecución de index.js"
 
🔸 Envía los cambios a GitHub:
 
git push origin main
 
✅ Este proyecto lo vas a seguir usando en las próximas lecciones. ¡No borres nada! Al final estará listo y con buenas prácticas de desarrollo.
